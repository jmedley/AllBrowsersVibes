# AllBrowsersVibes

The [AllBrowsers](https://medleyj64119.wixsite.com/allbrowsers) web site aggregates news and cutting edge web API reference for engineers. I'm building it as a means to learn AI and vibe coding. Early posts to the site are constructed using only LLM prompts. I am gradually converting the prompts to code using Copilot and Claude Code. 

The second part of this project is the [MDNDrafts](https://github.com/jmedley/MDNDrafts) repository. It holds drafts of reference documents for new web platform APIs. The drafts as they currently stand are suitable for publishing on MDN via commiting to the [mdn/content](https://github.com/mdn/content) repo.
